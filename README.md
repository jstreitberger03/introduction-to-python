# Introduction to Python

This repository contains materials for Python courses, organized into different modules.

## Structure

- **fma_course/**: Course materials focused on Financial Market Authority (FMA) context, covering basics, data structures, pandas, and visualization.

## Getting Started

1. Clone the repository.
2. Install the required dependencies (e.g., `pandas`, `numpy`, `matplotlib`, `jupyter`).
3. Navigate to the desired course folder and launch Jupyter Notebook/Lab.

# Introduction to Python

This repository contains materials for Python courses, organized into clear, beginner-friendly modules.

## Structure

- **fma_course/**: Main course in FMA context. It starts with absolute basics and grows toward data analysis, visualization, and practical automation.

## Getting Started

1. Clone the repository.
2. Install the required dependencies (e.g., `pandas`, `numpy`, `matplotlib`, `jupyter`).
3. Navigate to the desired course folder and launch Jupyter Notebook/Lab.

## Contents

### FMA Course (what you will learn)

#### Part 1: The Fundamentals
- **00_intro.ipynb** – **Start Here**: What Python is, how to use Jupyter Notebooks, and the "Zen of Python".
- **01_basics.ipynb** – **Core Skills**: Variables, math, printing text, and handling user input.
- **02_data_structures.ipynb** – **Organizing Data**: Lists, Dictionaries, Sets, and Tuples.
- **03_control_flow.ipynb** – **Making Decisions**: If/Else logic, Loops (For/While), and List Comprehensions.
- **04_functions_modules.ipynb** – **Reusable Code**: Writing functions, understanding scope, and importing modules.

#### Part 2: Data Analysis & Visualization
- **05_data_analysis_pandas.ipynb** – **Excel on Steroids**: Loading data, filtering rows, and calculating statistics with Pandas.
- **06_visualization.ipynb** – **Plotting**: Creating Line charts, Bar charts, and Histograms with Matplotlib.
- **07_practical_examples.ipynb** – **Real World**: Combining skills to solve problems like financial analysis and file automation.

#### Part 3: Robustness & Tools
- **08_error_handling_debugging.ipynb** – **Fixing Bugs**: Reading error messages, using `try/except`, and debugging strategies.
- **09_numpy_intro.ipynb** – **High-Speed Math**: Working with Arrays, Matrices, and Vectorized operations.
- **10_data_io.ipynb** – **Files**: Reading and writing Text, CSV, and JSON files safely.

#### Part 4: Advanced Topics
- **11_classes.ipynb** – **Object-Oriented Programming**: Creating your own custom data types (Classes and Objects).
- **12_numbers.ipynb** – **Under the Hood**: How computers store numbers (Bits, Bytes, and Floating Point errors).
- **13_ai_programming.ipynb** – **Modern Coding**: How to use AI assistants (like Copilot) effectively and responsibly.

#### Part 5: Data Science Essentials
- **14_sql_basics.ipynb** – **Databases**: Writing SQL queries (`SELECT`, `WHERE`) and loading results into Python.
- **15_eda_methodology.ipynb** – **Exploration**: A checklist for inspecting data quality, missing values, and distributions.
- **16_statistics_intro.ipynb** – **Proving It**: Hypothesis testing (T-Tests) and understanding Correlation vs. Causation.
- **17_machine_learning_intro.ipynb** – **Predicting**: The Scikit-Learn workflow (Train/Test Split, Linear Regression, Evaluation).

### For absolute beginners
- Start with `00_intro.ipynb` and `01_basics.ipynb` and run every cell.
- Tinker in the cells: change numbers/strings and observe the output.
- Run cells with `Shift+Enter`; restarting the kernel often fixes odd states.

### Next steps
- Launch `jupyter lab` or `jupyter notebook` in the `fma_course/` folder.
- Install missing packages if needed: `pip install pandas numpy matplotlib`.
